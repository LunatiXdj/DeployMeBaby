# **App Name**: X-Tool (PH-Service)

## Core Features:

- Enhanced Article Management: Adds a purchase price to the article management and displays Gross Profit Margin. Also provides an estimated amount for taxes payable based on sales revenue. The system acts as a tool by computing these quantities using formulas.
- Profit and Loss Statement Generation: Creates a dedicated section for generating a Profit and Loss (P&L) statement. Uses logged Project Data.
- Project Material Consumption Tracking: Allows material consumption lists to be linked to the project management module to record expenses. Creates a template for that list, formatted with project's data
- AI-Powered Report Optimization: Improves PDF and other report outputs. Utilizes AI, acting as a tool to determine which information and documents might be most relevant to include.

## Style Guidelines:

- Primary color: A calm and professional blue (#4682B4), to promote confidence in the CRM.
- Background color: A very light desaturated blue (#F0F8FF) creates a clean workspace.
- Accent color: A soft green (#90EE90) highlights positive actions such as saving.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral appearance.
- Code font: 'Source Code Pro' for displaying code snippets.
- Maintain a clean and structured layout, leveraging whitespace and clear section headings for improved readability.
- Use subtle transitions and animations to provide feedback on user interactions without being distracting.
